# Homebrew Tap

Custom Homebrew tap for my macOS apps.

## Install

```bash
brew tap momenbasel/tap
```

### PureMac

Open-source application manager and system cleaner.

```bash
brew install --cask puremac
```

### ConsoleMini

Turn a Mac mini into a living-room retro / PlayStation console launcher.

```bash
brew install --cask consolemini
```

> First launch: right-click the app in `/Applications` and choose **Open** (releases are not notarized yet).
